git中可以加入忽略文件权限的配置，具体如下：

$ git config core.filemode false  // 当前版本库
$ git config --global core.fileMode false // 所有版本库
这样就设置了忽略文件权限。查看下配置：

$ cat .git/config // 查看git的配置文件
